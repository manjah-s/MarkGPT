# Lesson 8: Key Takeaways and Summary

## Inverse RL Essentials

1. **Reward ambiguity**: Multiple rewards explain same behavior.
2. **Maximum entropy IRL**: Choose highest-entropy consistent reward.
3. **Feature matching**: Learning to match expert state visitation statistics.
4. **Apprenticeship learning**: Iteratively improve reward and policy.
5. **GAIL**: Adversarial trajectory distribution matching.
6. **IRL with deep learning**: Scale to high-dimensional observations.
7. **Preference learning**: Learn from comparisons rather than demonstrations.
8. **Causal structure**: Learn causal graphs underlying behavior.

## When to Use IRL

- Extract human values from behavior.
- Understand system objectives.
- Train agents aligned with human preferences.
- Learn complex implicit objectives.

## Practical Challenges

- Behavior underdetermines objectives (ambiguity).
- Biased behavior leads to biased learned rewards.
- Computational cost of IRL algorithms.
- Interpretability of learned rewards.
