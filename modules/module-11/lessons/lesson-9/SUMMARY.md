# Lesson 9: Summary

Transfer learning leverages knowledge from source tasks to accelerate target task learning. Fine-tuning, distillation, and multi-task learning are key mechanisms. Domain randomization and sim-to-real transfer enable simulation-to-reality deployment. Curriculum learning arranges tasks to progressively increase difficulty. Negative transfer warns against blindly applying transfer.

**Core Learning Outcomes:**
- Transfer dramatically accelerates learning when source-target align
- Fine-tuning requires careful hyperparameter tuning to avoid forgetting
- Multi-task learning enables emergent generalization
- Domain randomization enables sim-to-real transfer
- Curriculum learning provides gradual difficulty progression
- Negative transfer can occur with misaligned source-target pairs

For practitioners: Validate transfer works (target improves) before relying on it; use curriculum learning for complex tasks.
