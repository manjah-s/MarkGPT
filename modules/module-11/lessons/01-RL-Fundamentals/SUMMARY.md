# Lesson 1: Summary

Foundations of reinforcement learning rest on MDPs, value functions, and the Bellman equation. Q-learning enables agents to learn optimal behaviors from experiences. Understanding Markov properties, temporal differences, exploration-exploitation trade-offs, and value functions is essential before moving to complex RL algorithms.

**Core Learning Outcomes:**
- MDPs formalize RL problems mathematically
- Value functions decompose problems through Bellman equations
- Q-learning and TD learning enable practical learning
- Exploration is essential for discovering good policies
- Discount factors, state representations, and value approximation impact all RL algorithms

For practitioners: Implement Q-learning on simple gridworlds to build intuition before tackling complex domains.
